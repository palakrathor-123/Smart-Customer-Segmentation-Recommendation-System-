# Smart Customer Segmentation & Recommendation System

## Project Overview

This project groups mall customers into meaningful segments using clustering algorithms like:

- K-Means
- DBSCAN

and provides business recommendations for each customer segment.

---

## Features

- Upload CSV Dataset
- Data Preview
- K-Means Clustering
- DBSCAN Outlier Detection
- Silhouette Score
- Cluster Visualization
- Cluster Summary Table
- Recommendation Engine

---

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Matplotlib
